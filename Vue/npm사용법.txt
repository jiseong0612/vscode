우선 node js 홈페이지로 이동하여 노드를 설치한다.

>>> node -v  입력! 노드 버전을 확인하여 잘 설치됐는지 확인
>>> npm -v 입력! npm이 잘 설치 되었는지 확인(노드를 설치하면 같이 설치됨)
>>> 페키지 네임을 입력해야한다.. 야구게임에선 'number-baseball' 로 입력함 -->남의 소스를 패키지라 부른다..
>>> 끝날때 까지 엔터, 엔터, 엔터... 해줬음
>>> 위의 과정을 거치면 npm init을 한 폴더에 'package.json' 파일이 생긴다.
>>> npm i webpack webpack-cli -D  'D' : 개발할때만 쓰겠다. 'i' : install 약어, , webpack과 webpack-cli 두 개를 동시에 설치 
내가 남의것 몇 버전을 쓰고 있는지 확인하기 위해서 package.json 파일을 생생하는 것이다.

>>> npm install vue 입력! 이 명령어로 vue를 설치할 수 있다.
